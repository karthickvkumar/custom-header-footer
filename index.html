Fax and Document Automation Integrations

Concord provides an industry leading set of API's to allow simple, yet extremely comprehensive Fax and Document Automation Integrations for any possible deployment need.

Our API portfolio supports the following primary use-cases:

  • Outbound Fax delivery, real-time status and job reporting
  • Inbound Fax reception with image file and metadata for processing.
  • Document Automation for AI-driven document classification and field extraction (including barcode extraction)
  • Account Management for adds, moves, changes and deletes.

Additional API's are available on request. If you don't immediately find what you need, then please reach out to us and let us know what you need. We've almost certainly got a solution for you.
